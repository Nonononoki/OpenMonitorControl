This dialog selects the features to be reported. The features are specified as a group, and further refined by one or more filters on that group.

The feature groups are:
  • MCCS - All features defined in the MCCS specification. Includes features in the manufacturer specific range (xe0..xff) Note that feature definitions vary by MCCS version. ddcui depends on monitors correctly reporting their MCCS version.
  • Capabilities - All features defined in the capabilities string.
  • Scan - Examine all possible feature codes, except those known to be read-only or of type Table.
  • Manufacturer - Examine feature codes in the manufacturer specific range (xe0..xff)
  • Color - Color related feature codes.
The features reported can further be refined by the following filters. Not all filters apply to every group.
  • Include all features in capabilities string - Add all features in the capabilities string to those in the group.
  • Exclude features not in capabilities string - Include only features in the group that are also found in the Capabilities string
  • Show Unsupported Features - Include features in the group that are reported as unsupported.
  • Include Table Features - Include features of type Table. Unimplemented - cannot be checked.
